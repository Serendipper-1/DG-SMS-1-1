## **功能：**
1. 手机验证码获取X东CK，CK有效期默认一个月 
2. qq扫码获取X东CK，CK有效期默认一个月 
3. 对接青龙面板，多青龙选择上传
4. 对接xdd-plus

## 提醒一下
* 请选择性能好的设备运行，最好是云服务器或pc、x86软路由、oracle-arm(4-24)、群晖。
* 只支持青龙2.9+，2.8可能会出现无法上传问题。
* 本项目可以不依赖青龙运行，获取的CK需网页上手工复制
